Ubuntu Xenial (16.04)
ROS Kinetics
Pcl 1.7
tensorflow

ROS universal_robot package:

`git clone https://github.com/SyrianSpock/realsense_gazebo_plugin`

ROS car_demo autonomous vehicle package: 
`git clone https://github.com/cretaceous-creature/car_demo`
